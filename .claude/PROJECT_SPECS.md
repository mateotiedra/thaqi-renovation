# PROJECT_SPECS.md

## Project Overview

Thaqi Renovation is a portfolio/showcase website for a renovation business. The site targets potential customers looking to hire renovation services.

**Core Features:**
- Service listings - Pages describing renovation services offered
- Project gallery - Before/after photos of completed work
- Contact/Quote form - Customer inquiries and estimate requests

## Design Style

- Color scheme: Neutral base with bold accent color
- Aesthetic: Bold/impactful - strong typography, dramatic visuals
- Layout: Card-based grids for services and projects

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5 (strict mode)
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Linting**: ESLint 9 with Next.js core-web-vitals and TypeScript rules
